GUIDify

Version 1.0.0

About

GUIDify is a powerful Chrome extension that helps you generate multiple GUIDs (Globally Unique Identifiers) quickly and efficiently.

Features

  • Bulk Generation: Generate up to 100,000 GUIDs at once
  • Case Formatting: Choose between lowercase or UPPERCASE format
  • Auto Download: Automatically save GUIDs to a .txt file
  • Form Helper: Quick GUID upload button for file input fields
  • Persistent Settings: Your preferences are saved and synced
  • Unique Filenames: Timestamp-based filenames prevent overwrites

How to Use

  1. Click the extension icon in your toolbar
  2. Enter the number of GUIDs you want to generate (1-100,000)
  3. Choose your preferred case format (lowercase/UPPERCASE)
  4. Click "Generate" button
  5. The file will be automatically downloaded to your Downloads folder

Form Helper

When enabled in settings, a "Load GUIDs" button appears next to file upload fields on web pages. Click it to generate and upload GUIDs directly to the form.

Technical Details

GUID Format: RFC 4122 compliant UUIDs (version 4)

Example: a1b2c3d4-e5f6-4789-a012-b3c4d5e6f7a8

Randomness: Uses cryptographically secure random number generation

Privacy

This extension does not collect, store, or transmit any personal data. All GUID generation happens locally in your browser. Your settings are stored locally using Chrome's sync storage.

Permissions

  • Downloads: Required to save generated GUIDs as .txt files
  • Storage: Required to save your preferences (case format, form helper toggle)
